Math Connection

The balance grows according to an exponential model:

y = a(b)x

  • a = starting balance
  • b = growth factor (interest)
  • x = number of months

Each month the balance multiplies by: b = 1 + (APR / 12)

Example: If APR = 24%
Monthly rate = 0.24 / 12 = 0.02
Growth factor: b = 1.02

That means the balance follows:
Balance ≈ a(1.02)x
